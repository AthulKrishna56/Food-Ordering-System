# 🍽️ Food Ordering System

A Java-based **Food Ordering System** developed using **NetBeans** with **MySQL** as the database. The application allows users to browse food categories, place orders, and manage customer order details through an easy-to-use interface.

## Features

* User-friendly Java GUI
* Food category management
* Food item management
* Place customer orders
* Order history and details
* Automatic total bill calculation
* MySQL database integration

## Technologies Used

* Java
* Java Swing
* NetBeans IDE
* MySQL
* phpMyAdmin
* XAMPP
* JDBC

## Database

The project uses a MySQL database named **food_ordering**.

Tables:

* `categories`
* `food_items`
* `orders`
* `order_items`

## Software Requirements

* NetBeans IDE
* Java JDK 8 or later
* XAMPP
* phpMyAdmin
* MySQL
* JDBC Driver (MySQL Connector/J)

## Installation

1. Install Java JDK and NetBeans.
2. Install XAMPP and start **Apache** and **MySQL**.
3. Open **phpMyAdmin** and create a database named `food_ordering`.
4. Import the provided SQL database file.
5. Open the project in NetBeans.
6. Update the database connection details if required.
7. Build and run the project.
